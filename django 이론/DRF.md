# Django Rest Framework

- DRF는 장고를 기반으로 웹 API를 구축할 수 있도록 기능 등을 만들어 놓 은 툴킷
- RESTful한 API 형태의 기능을 제공

## 장점
- API, 개발을 쉽게 만들어줌
- 인증 정책 OAuth1, OAuth2 사용 가능
- Serializer(직렬화) 기능을 제공 (Model > JSON, JSON > Model)
- 문서화, 커뮤니티 지원

## 핵심 요소
- 요청 / 응답
- View와 ViewSet
- 라우터
- 직렬화
- 인증 / 권한
- 페이징 / 필터
